卷十五禮十(賓禮 )
◎優禮元裔
洪武三年，封密迪哩巴拉(舊作賣地里八喇。 )為崇禮侯。誥曰：「昔帝王之有天下，必封前代子孫，使作賓王家，其來尚矣。元失其馭，四海紛爭。朕削平群雄，混一區宇，為天下主。而密迪哩巴拉為元之宗孫。比者遣將北征，爾祖已殂。既克應昌，爾乃來歸。朕念帝王之後，爰稽古制，錫以侯封，稱朕優禮之意。」尋賜第龍光山。(《聖政記》。 )七年九月，上謂廷臣曰：「草木無心，遇春而茂，遇秋而零，氣之所感，猶知榮悴，況於人乎？崇禮侯密迪哩巴拉南來已五載，今已長成，豈無父母鄉土之思？宜遣之還。」於是厚禮而歸之。(《通典》。 )
◎蕃王來朝洪武二年九月壬子，定蕃王、蕃使朝貢禮。(《昭代典則》。 )二十七年四月，更定蕃國朝貢儀。(《大政記》。 )
永樂六年八月，浡泥國王麻那惹加那帥妃及弟妹子女陪臣泛海來朝。次福建。遣中官往迎勞。所過州縣皆宴。比至，帝慰勞再三，饗於奉天門。妃以下饗於他所。禮訖，送入會同館。禮官請：「王見親王儀？」帝令准公侯禮。
九年，滿剌加國王拜里迷蘇喇帥妻子陪臣五百四十餘人來朝。有司供張會同館，入朝奉天殿。帝親宴之。妃以下宴他所。光祿日致牲牢上尊，賜賞甚厚。及歸，賜宴奉天門。禮官餞於龍江驛。復賜宴龍潭驛。十七年，王子母干撒於的兒沙率妻子陪臣來朝謝恩。
十年九月，浡泥國王遐旺偕其母來朝。命禮官宴之會同館。光祿寺旦暮給酒饌。明日，帝饗之奉天門。王母亦有宴。越二日再宴。
十五年，西洋蘇祿國東王、西王、{山同}王帥其家屬頭目，凡三百四十餘人，浮海來朝。比辭歸，東王次德州，卒於館。帝遣官營葬，賜謚曰「恭定」。
十八年八月，古麻喇朗國王干義奔敦帥妻子陪臣來朝。還至福建卒。謚曰「康靖」，葬以王禮。
宣德八年，滿剌加王西里麻哈剌率妻子陪臣來朝。抵南京，天已寒。命俟春和北上，別遣人齎敕勞賜王及妃。及入朝，宴齎如禮。(已上《外國傳》及《通典》。 )
◎遣使之蕃
明祖既定天下，分遣使者，奉詔書往諭諸國，或降香幣以祀其國之山川。(《禮志》。 )
洪武二年，命侍讀學士張以寧等，往對陳日奎為安南國王。以寧抵安南境，日奎先卒。其侄日奎嗣立，迎請誥印。以寧不與，諭日奎請命於朝。以寧駐安南俟命，教世子服三年喪，令其國人效中國行頓首、稽首禮。帝聞而嘉之，賜璽書比之陸賈、馬援。安南與占城構兵，帝命編修羅復仁等諭令罷兵，兩國皆奉詔。(《外國傳》，《張以寧傳》。 )
十二年二月，定遣使外國儀注。(《集禮》。 )
宣德六年，禮部侍郎章敞使安南，命黎利權國事。利遣人白相見禮。敞曰：「汝敬使者，所以尊朝廷。奚白為？」利聽命，趨拜，下坐。啖以聲色，不為動。還致厚贐，不受。利以貢使。及關，悉閱貢物。封其贐，付關吏。(《章敞傳》。 )
行人侯使交址。關門卑，前驅傴而入。叱曰：「此狗竇耳，奈何辱天使？」交人為毀關，乃入。及歸，饋遺無所受。(《侯傳》。 )
景泰元年六月，以給事中李實為禮部侍郎，大理寺丞羅綺為少卿，令齊敕書諭衛拉特君臣，遂偕阿剌使俱往。(《三編》。 )七月，遣右都御史楊善等使衛拉特。(同上。 )
成化四年，遣太監鄭同、崔安使朝鮮。遼東巡按御史侯英奏同等所過驛騷狀。因言：「同、安俱朝鮮人。見其國王，不免屈節，褻中國體。乞寢成命。或翰林或給事中、行人內，推選一員往使，為便。」帝然之。自後，賞賜遣內臣；冊封正副使選廷臣有學行者充。(《朝鮮傳》。 )
安南多寶貨，使者率從水道，挾估客往以為利，交人頗輕之。弘治元年，侍講劉戩頒詔安南，由南寧乘傳抵其國。交人大驚。戩依舊制，受陪臣拜謁。不交一語，越宿即行。饋遺一無所受。使人要於塗，固致之，卒麾去。(《章敞傳》。 )
嘉靖十八年，時當頒詔朝鮮。帝以安南事未決，欲因以往覘，命擇大臣有學識者以往。乃起黃綰禮部尚書兼翰林學士為正使，諭德張治副之。(《黃綰傳》。 )
隆慶時，給事中魏時亮使朝鮮。故事：王北面聽詔，使者西面。時亮力爭，乃南面宣詔。(《時亮傳》。 )
○蕃使入貢
洪武二年六月，安南國王陳日奎遣使奉表來朝，貢方物。帝喜，賜宴。
八月，高麗國王王顓遣使修貢，請封。命符璽郎契斯齎詔及金印往封之。
十二月，阿答阿者遣使奉表，貢象、虎、方物、命中書省管句、甘桓等往封為城國王，並賜彩幣，頒《大統歷》。
四年，占城入貢。言：安南侵擾。因求兵器。帝命諭之曰：「占城、安南同奉正朔，乃擅自構兵，妄害生靈。既失事君之禮，又乖交鄰之道。已咨安南國王，令即日罷兵。王亦宜講信修睦，各保疆士。所請兵器，於王何吝？但兩國互構，而賜占城，是助王相攻，甚非撫安之義。」
是年，日本遣僧祖來奉表稱臣，貢方物。太祖嘉之，宴賚其使者。(已上《外國傳》。 )
五年二月，安南陳叔明遣使入貢。(時殺其主日堅而自立。 )主客曹已受表；曾魯取副封視之，白尚書。詰使者：「前王日堅，今何驟更名？」使者不敢諱，具言其實。帝曰：「島夷乃狡獪如此耶！」卻其貢。(《曾魯傳》。 )
七年，諭中書及禮部臣曰：「古諸侯於天子，比年一小聘，三年一大聘。九州之外，則每世一朝。所貢方物，表誠敬而已。惟高麗頗知禮樂，故令三年一貢。他遠國如占城、安南、西洋瑣里、瓜哇、浡泥、三佛齊、暹羅斛、真臘諸國，入貢既頻，勞費太甚。今不必復爾。其移牒諸國，俾知之。」
呂宋、瑣里諸國來貢。帝謂中書省臣曰：「西洋諸國，素稱遠番。涉海而來，難計歲月。無論疏數，『厚往而薄來』，可也。」
高麗貢使失風，洪師范等二十九人溺焉。帝憫之，謂中書省曰：「高麗貢獻繁數，既困敝其民，而涉海復虞覆溺。宜遵古諸侯之禮，三年一聘。貢物惟所產，毋過侈。其明諭朕意。」
十二年，高麗貢黃金百斤，白銀萬兩。以不如約，卻之。十三年，日本遣使來貢，無表，卻之。
十七年，高麗遣使貢馬二千匹，言：「金非地所產，願以馬代輸。余皆如約。」許之。尋以高麗既聽命，宜損其貢數。令三年一朝，貢馬五十匹。(已上《外國傳》。 )
二十六年，定：凡四夷歸化人員及朝貢使客初至，會同館主客部官隨即至彼，點視正從一切，處分安妥。仍加撫綏，使知朝廷恩澤。(《會典》。 )
二十七年，安南遣使由廣東入貢。帝怒，遣官詰責，卻其貢。
永樂元年，日本入貢。禮部尚書李至剛奏：「故事：番使入中國，不得私攜兵器鬻民。宜敕所司核其舶；諸犯禁者悉籍送京師。」帝曰：「外夷貢，履險蹈危，有所鬻以助資斧，豈可概拘以禁令？至其兵器，亦準時直市之，毋阻其向化心。」
西洋瑣里、剌泥諸國來貢，因攜胡椒與民市。有司請徵其稅，帝曰：「徵稅以抑逐末之民，豈以為利？今遠人慕義來，乃取其貨。所得幾何，虧損國體多矣。其勿徵。」(已上《外國傳》。 )
七年，威剌特遣使貢馬，請封。遂封嗎哈穆特為順寧王，太平為賢義王，巴圖博為安樂王。(《三編》。 )
九年九月，朝鮮使臣將歸，便有賜齎。禮部尚書趙Р不以奏。帝怒曰：「是且使朕失遠人心。」遂下之獄。(《外國傳》。 )
阿嚕台復來貢馬，因請得役屬吐番諸部；求朝廷刻金作誓，磨其金酒中，飲諸酋長以盟。眾議欲許之，因黃淮言，乃止。(《黃淮傳》。 )
十三年正月丁未，馬哈木等貢馬謝罪，且還前所留使，詞甚卑順。帝曰：「衛拉特故不足與較。」受其獻，館其使者。(《外國傳》。 )
宣德末，古里、蘇門答剌等十一國貢使，久留京師未遣。正統元年閏六月，命與瓜哇使臣同行。敕瓜哇國王，加意撫恤，分遣還國。(同上。 )
七年十一月，衛拉特額森遣使入貢。故事：衛拉特使止三十餘人。其後利朝廷賞齎，所遣動以千計。大同供應費至三十餘萬。是春帝以貢使太多，限三百人入關。及秋至大同者，復二千餘人。以馬易弓，藏於衣篋，不可勝計。巡撫羅亨信請於居庸關詰檢之，不許。郭敬歲造箭鏃數十瓮遺其使，帝亦不問。(《三編》。 )
正統十四年十月，衛拉特托克托布哈遣使入貢。帝從胡氵熒、王直議，厚賞賜以間之。(同上。 )
天順五年，保喇遣使來貢，受約。又請改大同舊貢道，由陝西蘭縣入。許之。(河套之患始此。保喇舊作孛來。 )
成化三年正月，朝鮮頻貢異物、海青、白鷳之屬。敕國王李柔常貢，勿事珍奇。
四年夏，日本遣使貢馬謝恩，禮之如制。十一月，復遣使臣清啟入貢。其從者殺人於市，有司請治罪。清啟奏請帶回本國，如法論治；且自服不能鈐束之罪。上命俱赦之。自是使者益無忌。
六年四月，申諸番三歲入貢之例；多不過百五十人，由四川雅州入。國師以下不許貢。於是貢使漸稀。
十七年九月，暹羅貢使還。至中途，竊賣子女，且多載私鹽。命遣官戒諭諸番。(已上《外國傳》。 )
十九年十月，西番賽瑪爾堪(舊作撒馬兒罕 )等國，共貢獅子，奏請大臣往迎。職方郎如陸容言：「此無用之物，在郊廟不可為犧牲，在乘輿不可備驂服，宜勿受。」周洪謨亦言：「往迎非禮。」帝卒遣中使往迎之。(《通紀》。 )
弘治元年六月，小王子遣使千五百餘人，款關求通貢。巡撫許進以便宜納之，聞於朝。所上書，自稱「大元汗」。朝廷方務優容，許遣五百人詣京師。(《三編》。 )
十一月，土魯番阿哈瑪特遣使入貢。稱與哈尚締姻，乞賜蟒袍及九龍渾金膝衤闌諸物。使至甘州，哈密都指揮阿穆呼朗告變。朝廷亦不罪番使，但令還諭其主，反侵地。阿哈瑪特竟不奉命，復遣使來貢。禮官議：薄其賞而拘其使臣。(《外國傳》及《三編》。 )
二年十二月，賽瑪爾堪由廣東貢獅子鸚鵡等物。禮部尚書耿裕等言：「南海非西域貢道，請卻勿受。」禮科給事中韓鼎等亦言：「猙獰之獸，狎玩非宜。且騷擾道路，供費不貲，不可。」從之。(《外國傳》。 )
五年九月，西域呼喇濟(舊作火剌扎。 )國回回布魯完(舊作怕魯灣 )等由海貢玻璃、瑪瑙諸物。帝不納，賜道里費，遣還。(《西域傳》。 )
舊設南北兩會同館，接待番夷使客。遇有貢夷到京，主客司、員外郎、主事輪赴會同館、點視方物，譏防出入。弘治五年，各夷來貢者眾，添設提督會同館主事一員。(《會典》。 )
六年，土魯番貢使留京師，頻有宣召。禮部尚書耿裕等言：「番人不道，因朝貢許其自新。彼復潛稱可汗，興兵犯順。陛下憂假其使，適遇倔強之時，彼將謂天朝畏之，益長桀驁。」帝即遣其使還。(《耿裕傳》。 )
嘉靖二年，禮官言：「諸國使臣在途者遷延隔歲，在京者伺候同賞。光祿、郵傳，供億不貲，宜示以期約。」因列上禁制數事，從之。(《西域傳》。 )
三年，魯迷國貢獅子、西牛、西狗、西馬及珠、玉諸物。給事中鄭一鵬引漢閉玉門關謝西域故事，請敕邊臣，量行賞齎，遣還國，以彰不寶遠物之德。不聽(鄭《一鵬傳》。 )
七年六月，土魯番求通貢，乞歸羈留使臣。提督三邊都御史王瓊再疏請。詔還番使，通貢如故。(《王瓊傳》。 )
十二年，賽瑪爾堪偕天方、土魯番入貢。稱王者至百餘人。禮官夏言等言：「土魯番十五王，天方二十七王，賽瑪爾堪五十三王，實前此所未有。弘治間，回賜敕書，止稱一王。若循賽瑪爾堪往歲故事，類答王號，人與一敕，非所以尊中國制外蕃也。」帝納其言，國止給一敕。且加詰讓，示以國無二王之義。
十五年，甘肅巡撫趙載奏：「諸國稱王者，至一百五十餘人，皆非本朝封爵，宜令改正。且定貢使名數。通事宜用漢人，毋專用色目人，致交通生釁。」部議從之。(已上《西域傳》。 )
十八年，日本貢使至寧波，守臣以聞。時不通貢者已十七年。敕巡按御史督同三司官核實，果誠心效順，如制遣送，否則卻回。且嚴居民交通之禁。(《外國傳》。 )
二十六年三月，甘肅巡撫楊博言：「西域入貢人多，宜為限制。」禮官言：「神宗故事：惟哈密每年入貢，貢使三百人；送十一赴京，餘留關內，有司供給。他若哈烈哈三、土魯番、天方、賽瑪爾堪諸國，道經哈密者，或三年五年一貢，止送三、五十人，其存留賞齎，如哈密例。宜敕邊臣恪遵。」制：「可。」(《西域傳》。 )
二十七年，日本周良復求貢。巡撫朱紈以聞。禮部言：「日本貢期及人與船數雖違制，第表辭恭順。若概加拒絕，則航海之勞可憫；若稍務含容，則宗設、素卿之事可鑑。宜敕紈起送五十人，餘留嘉賓館，量加犒賞，諭令歸國。」報可。(《外國傳》。 )
三十九年，朝鮮國陪臣到京。命禮部尚書待。又泰寧等夷人頭目，命總督戎政勛臣待。自後夷使至京，以此為例。(《會典》。 )隆慶四年十一月，諳達遣使乞封貢。(《三編》。 )
《通典》論曰：「謹按漢時呼韓邪單于入覲。蕭望之謂：『單于非正朔所加，宜待以不臣之禮。』後代蕃使朝貢儀注，率多憂假，皆本於望之之議。明初，定蕃使入朝，宣制撫問國君，並及使者，略如敵國之儀。此亦沿習歷代具文。而揆諸大一統之朝，所以駕馭蕃服者，其說固未當也。夫普天率土，莫非王臣。要荒之裔，以職事來朝，正宜示以彝憲，俾知共主之尊，義在則然；即頑梗不率之風，因之盡戢。彼望之之議，特以威德不能及遠，聊為權宜之計耳！後儒無識，遂至承訛襲謬。豈知帝王撫臨中外，予以拊循，尤當明其體制。若概以敵國待之，轉致取輕殊俗。於恩威並用之道，寧有當哉？」
