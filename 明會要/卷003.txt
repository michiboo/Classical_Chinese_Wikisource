洪武元年正月，立世子標爲皇太子。(《本紀》。 )
太子文奎，建文元年，立爲皇太子。燕師入，七歲矣，莫知所終。(《諸王傳》。 )
永樂二年四月甲戌，立子高熾爲皇太子。帝初起兵，高煦常從戰有功，帝以爲類己；高煦亦以此自負，謀奪嫡。及議建儲，邱福等言：「高煦有功宜立。」獨金忠力爭以爲不可。帝召解縉問之。縉稱：「皇長子仁孝，天下歸心。」帝不應。縉又頓首曰：「好聖孫。」謂皇孫瞻基也。復問黃淮、尹昌隆，對與縉同。帝意乃決。
二十二年，仁宗即位。十月壬子，立子瞻基爲皇太子。明年五月，帝不豫，遣使召皇太子於南京。太子方謁孝陵，即日就道。時南京頗傳凶聞，又傳漢王高煦謀伏兵於道，邀太子。羣臣請整兵衛，或請從間道行。太子不可，曰：「君父在上，誰敢幹之。」驛道馳還，入宮發喪，越十日即位。
宣德三年二月戊午，立皇長子祁鎮爲皇太子。時胡皇后未有子，而孫貴妃有寵，陰取宮人子爲己子。帝以長子生，大喜。甫八日，羣臣上表請立爲太子。皇后亦屢表請早定國本。貴妃佯驚曰：「後病痊自有子，吾子敢先後子耶？」帝不允。貴妃子遂立。(已上《三編》。 )
正統十四年八月己巳，皇太后詔立皇子見深爲皇太子。
景泰三年五月甲子，廢皇太子見深爲沂王，立皇子見濟爲皇太子。四年十一月辛未，皇太子見濟薨，諡「懷獻」。
天順元年三月己巳，復立長子見深爲皇太子。
成化七年十一月甲寅，立皇子祐極爲皇太子。八年正月薨，諡「悼恭」。(已上《本紀》。 )
十一年十一月癸丑，立皇子祐樘爲皇太子。時皇子潛養西內，已六年。是年五月，遣使至安樂堂迎皇子。越數日，帝召閣臣問曰：「皇子既出，將何以處之？」商輅曰：「陛下踐祚十年，儲副未立，天下引領望久矣。當即立爲皇太子，安中外心。」帝遂定名，頒詔中外。(《三編》。 )
宏治五年三月戊寅，立皇子厚照爲皇太子。
嘉靖十八年正月庚子朔，立皇子載壑爲皇太子。乙卯，幸承天，太子監國。(年四歲。 )二十八年三月丁亥，皇太子薨，諡「莊敬」。
隆慶二年三月辛酉，立皇子翊鈞爲皇太子。(已上《本紀》。 )
萬厤二十九年十月辛卯，立皇長子常洛爲皇太子。時太子年二十，羣臣屢請冊立、冠、婚並行。沈一貫草敕，請下禮官具儀。而廷議有欲先冠、婚，後冊立者。一貫不可，曰：「不正名而苟成事，是降儲君爲諸王也。」帝意亦悟，命即日舉行。既而復令改期，一貫封還諭旨力爭，乃立爲皇太子。(《三編》。 )
太子慈烺，莊烈帝第一子，崇禎二年二月生，九月立爲皇太子。京師陷，賊獲太子，僞封宋王。及賊敗西走，太子不知所終。由崧時，有自北來稱太子者。驗之，以爲駙馬都尉王昺孫王之明者僞爲之，繫獄中。南京士民譁然不平。袁繼咸、劉良佐、黃得功輩皆上疏爭。左良玉起兵，亦以救太子爲名。一時真僞莫能知也。(《諸王傳》。 )
洪武元年正月辛巳，中書省都督府請倣元制，以太子爲中書令。帝以元制不足法，令詹同考厯代東宮官制，設少師、少傅、少保、詹事、率府使、諭德、贊善、賓客等官，以李善長兼領之。諭之曰：「朕於東宮，不別設府僚，而以卿等兼領者：蓋軍旅未息，朕若有事於外，必太子監國；若設府僚，卿等在內，事當啟聞；太子或聽斷不明，與卿等意見不合，卿等必謂府僚導之，嫌隙易生。朕所以特置賓客、諭德者，欲輔太子德性，且選名儒爲之，職此故也。昔周公教成王克詰戎兵，召公教康張皇六師，此居安慮危，不忘武備。蓋繼世之君，生長富貴，暱於安逸，不諳軍旅。一有緩急，罔知所措。二公之言，其並識之。」(《洪武聖政記》。 )
十一月辛丑，建大本堂，取古今圖籍充其中，徵四方名儒教太子諸王，選才俊之士充伴讀。上時臨幸，評論古今。嘗御文樓，太子侍，問近與諸臣讀何史？對曰：「漢七國事。」問曲直安在？對曰：「曲在七國。」上曰：「此講官一偏之說。景帝爲太子時，以博局殺吳世子；及爲帝，又輕聽晁錯，黜削諸侯。七國之變，實由於此。若爲諸子，則宜言『藩王當上尊天子，毋撓天下公法』。如此，則爲太子者知敦睦九族，隆親親之恩；爲諸子者，知夾輔王室，盡君臣之義矣。」(《三編》。 )
十年六月丙寅，命羣臣，大小政事，先啟皇太子，裁決上聞。諭曰：「自古創業之君，厯涉勤勞，達人情，周物理，故處事咸當。守成之君，生長富貴，若非平昔練達，鮮不謬者。故吾特命爾日臨羣臣，聽斷諸司啟事，以練習國政。」(《興宗傳》。 )
永樂二年三月，《文華寶鑑》成，上御奉天門，召皇太子授之。又顧大學士解縉等曰：「朕皇考訓戒太子，嘗集經傳格言爲書，名《儲君昭鑑錄》，今朕此書稍充廣之。昔秦始皇教太子以法律，晉元帝授太子以非聖之書，帝王之道，廢而不講，所以速亡。朕此書皆大經大法。卿等兼輔東宮，從容閒暇，以此陳說，庶幾成其德業，他日不失爲守成令主。」(《昭代典則》。 )
成祖數北征，命太子監國，裁決庶政。高煦、高燧與其黨日伺隙讒構。或問太子，亦知有讒人乎？曰：「不知也。吾知盡子職而已。」十年，北征還，以太子遣使後期，且書奏失辭，悉徵宮僚黃淮等下獄。十五年，高煦以罪徙樂安。明年，黃儼等復讒太子擅赦罪人。宮僚多坐死者。侍郎胡濙奉命察之，密疏誠敬孝謹七事以聞。其後，黃儼等謀立高燧，事覺伏誅。高燧以太子力解得免。自是太子始安。(《本紀》。 )
永樂十二年，北征還，悉徵東宮官屬下獄。以金忠勳舊不問，而密令審察太子事。退言「無有」。帝怒，忠免冠頓首流涕，願連坐以保之。以故，太子得無廢。(《金忠傳》。 )
十八年十二月，皇太子奏：「前過山東境內，遇民饑，即令布政使放粟賑之。」帝曰：「是也。昔范仲淹之子猶能舉麥舟濟其父之故舊，況百姓吾之赤子乎？」(《大訓記》。 )
景泰三年四月，賜閣臣陳循、高穀白金各百兩，江淵、王一甯、蕭鎡、商輅半之。上自即位後，久欲易皇太子，以己子見濟代之，而難於發言，遲回久之。太監王誠、舒良爲上謀，先賜閣臣以緘其口，然猶未發也。會廣西土目黃竑有罪下獄，遣其黨千戶袁洪至京師行賂。有教之上書迎合帝意者，乃倡「易儲議」以上。上曰：「萬里之外，乃有此忠臣！」趣下廷臣議。於是更封太子爲沂王，立見濟爲太子。(《三編》。 )
四年，懷獻太子既薨，中外望復沂王於東宮。御史鍾同與禮部郎中章綸早朝，語及沂王，皆泣下，因約疏請復儲。同上疏抗論時政，遂及復儲事，其略曰：「父有天下，固當傳之於子。乃者太子薨逝，足知天命有在。臣竊以爲：上皇之子，即陛下之子。沂王天資厚重，足令宗社有託。伏望擴天地之量，敦友於之仁，蠲吉具儀，建復儲位，實萬世無疆之休。」(《鍾同傳》。 )
五年七月，南京大理寺少卿廖莊上疏，略曰：「太子者，天下之本，上皇之子，陛下之猶子也。宜令親儒臣，習書策，以待皇嗣之生。使天下臣民曉然知陛下有公天下之心，豈不美與？蓋天下者，太祖、太宗之天下。仁宗、宣宗繼體守成者，此天下也。上皇北征，亦爲此天下也。今陛下撫而有之，宜念祖宗創業之艱難，思所以繫屬天下之人心。即弭災召祥之道，莫過於此。」(《廖莊傳》。 )
天順二年四月，太子始講學於文華殿。(即所謂左春坊也。 )
八年正月乙卯，帝不豫，臥文華殿。有間太子於帝者，帝密召李賢告之。賢頓首伏地曰：「此國本也。願陛下三思！」帝曰：「然則必傳位太子乎？」賢又頓首曰：「宗社幸甚。」帝起，召太子至。賢扶太子令謝。太子抱帝足泣，帝亦泣。讒竟不行。(已上《三編》。 )
憲宗一日視內藏，見累朝金窖俱盡，謂太監梁芳及韋興曰：「靡費內藏，實由汝二人。」興不敢對。芳曰：「建顯靈宮及諸祠廟，爲陛下祈萬年福耳！」帝不懌曰：「吾不汝瑕，後之人將興汝計矣。」芳大懼，遂說貴妃，勸帝廢太子而立興王。會泰山屢震，占者言：「應在東朝。」帝懼乃止。(《梁芳傳》。 )
宏治七年正月，馬文升以太子年及四齡，當早諭教，請擇𨣳謹老成，知書史，如衛聖楊夫人者，保抱扶持。凡言語動止，悉導之以正。若內庭曲宴、鍾鼓司承應、元宵鼇山、端午競渡諸戲，皆勿令見。佛老之教，尤宜屏絕，恐炫惑心志。帝深納之。(《馬文升傳》。 )
隆慶五年二月甲午，廷臣及朝覲官謁太子於文華左門。(《本紀》。 )
萬厤十年八月丙申，皇長子常洛生，恭妃王氏出也。十四年，鄭貴妃生子常洵，遂進封，而王妃生皇長子已五歲，不益封。中外藉藉，疑帝將立愛。戶科給事中姜應麟抗疏言：「欲別嫌明微，莫若冊立元嗣爲東宮，以定天下之本。」帝怒，貶應麟廣昌典史。已，吏部員外郎沈璟、刑部主事孫如游繼言之，並得罪。兩京申救者，疏數十上，皆不省。自後言者蠭起。(《姜應麟傳》。 )
二十二年，皇長子常洛已十四歲。帝手諭閣臣，令議出閣講學禮儀，免御門慶賀，用輔臣侍班，侍臣六人侍講讀，俱如東宮儀。大學士王錫爵言：「皇長子出閣，輔臣每日一員侍班，此雖隆慶年新例，但今天顏尚不得時覲，而於皇長子進見頻數，似涉嫌疑。請自初講二日以後，間十日輪侍。」從之。(《三編》。 )
三十一年，獲妖書，託鄭福成爲問答。鄭福成者，謂鄭之福王當成也。言帝必易東宮，詞極詭妄。帝大怒，敕錦衣衛𢯱捕甚急。久之，乃得曒生光者，坐極刑。四十三年，梃擊事起。主事王之寀疏言張差獄情，詞連貴妃宮內侍龐保、劉成等。朝議洶洶。貴妃聞之，對帝泣，帝曰：「須自求太子。」貴妃向太子號訢。帝召見羣臣，令太子降諭禁株連。於是張差獄乃定。(《后妃傳》。 )
洪武二十五年夏四月，皇太子標卒。帝御東角門，召對羣臣，慟哭曰：「太子不幸至此！古云：『國有長君，社稷之福。』朕意欲立燕王，何如？」學士劉三吾進曰：「皇孫世嫡承統，禮也。即立燕王，置秦晉二王於何地？」帝意遂決。九月，立孫允炇爲皇太孫。(《三編》。 )
二十六年七月戊申，選秀才張宗濬等，隨詹事府官分直文華殿，侍皇太孫。
初，太祖命太子省決奏章。太子仁厚，於刑獄多所減省。至是以命太孫，亦復佐以寬大。嘗請於太祖，遍考《禮經》，參之厯朝刑法，改定《洪武律》畸重畸輕者七十三條。(已上《本紀》。 )
帝命太孫裁決庶務。邏者獲盜七，太孫目之，言於帝曰：「六人者盜，其一非是。」訊之果然。帝問：「何以知之？」對曰：「《周禮》聽獄，色聽爲先。此人眸子瞭然，顧視端詳，必非盜也。」帝曰：「治獄貴通經，信然。」(《三編》。 )
永樂九年十一月，立長孫瞻基爲皇太孫。
皇太孫始冠，自是巡幸征討皆從。嘗命學士胡廣等即軍中爲太孫講論經史，每語仁宗曰：「此他日太平天子。」(已上《本紀》。 )
